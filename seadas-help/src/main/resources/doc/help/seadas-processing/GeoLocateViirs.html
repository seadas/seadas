Details

geolocate_viirs creates a GEO file from VIIRS L1A input files.

For geolocate_viirs to run properly, it needs the DEM files. The DEM files for VIIRS are on NASA OBPG FTP server:

ftp://samoa.gsfc.nasa.gov/pub/VIIRSL1/DEM-LWM.tar.gz

Extract the tar file in the $OCSSWROOT/run/data/viirs directory. The tar file is 5G, and the extracted file size will be 52G.
Point the terrain_path to the location of the extracted tar file.
