SeaDAS 7.1 Help

SeaDAS is a scientific data visualisation, analysis, and processing software application for remote-sensing satellite data which derives from and is an extension of the BEAM Earth Observation Toolbox and Development Platform. SeaDAS includes support for the data products produced by NASA's Ocean Biology Processing Group (http://oceancolor.gsfc.nasa.gov). This support includes data processing capabilities to allow users to process data between the various levels (i.e. Level 0 through Level 3).

Please be aware that some of the component-level SeaDAS 7.1 help pages have not yet been revised to match the current version. For the latest SeaDAS help and tutorials see http://seadas.gsfc.nasa.gov


Component Reference

Where available, component names are linked to their respective help pages.

File Tools

File Manager

File -> File Manager   (Docked: left-side)
Displays all opened files in a tree view.

General Processing Tools

Collocate

Processing -> Collocate   (Toolbar: Processing)
Collocates two files together into a new file (useful for parameter comparisons, etc.)

Crop

Processing -> Crop   (Processing Toolbar)
Crops the file spatially to create a new file (default boundaries are the current view).

Flip Data

Processing -> Flip Data
Applies a data flip to create a new file

Mosaic

Processing -> Mosaic   (Toolbar: Processing)
Creates a mosaic file from multiple files.

Orthorectify

Processing -> Orthorectify
Applies orthorectification to create a new file.

Reproject

Processing -> Reproject
Applies reprojection to create a new file.

Band Tools

DEM Band

Creates DEM elevation and optionally DEM-corrected lat/lon bands from a selected DEM.

Filtered Band

Creates a filtered band from the currently selected band.

Math Band

Create a new band using an arbitrary mathematical expression.

Real Band

Computes a "real" band from a virtual or filtered band.

Mask Tools

Bathymetry

Add elevation, bathymetry, and topography bands and bathymetry mask.

Coastline, Land & Water

Add coastline, land and water masks

Magic Wand

Creates a ROI mask using a magic wand.

Mask Manager

Manage binary data masks and ROIs (regions of interest).

Color Tools

Colorbar

View and/or Export the colorbar.

Color Manager

Set the color palette of band image.

RGB Image

Creates a 3-band composite image using red, green, and blue color palette.

Layer Tools

World Map Layer

Pin Placer
