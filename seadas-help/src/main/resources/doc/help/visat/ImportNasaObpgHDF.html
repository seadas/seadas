Import NASA Ocean Color Product

The NASA Ocean Biology Processing Group (OBPG) provides a consistently processed set of ocean color data products from several missions, including MODIS (Aqua and Terra platforms), SeaWiFS, OCTS and CZCS. For more information on the data and for data access, please refer to the NASA Ocean Color Web .

SeaDAS supports the all of the data products produced by the OBPG, including:

  • Level 1 through Level-3 Data and Browse imagery for:
    • CZCS
    • OCTS
    • SeaWiFS
    • MODIS
    • VIIRS
    • Aquarius
  • Limited support for:
    • MOS
    • OSMI
    • OCM-1
    • OCM-2
