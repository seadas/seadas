About SeaDAS

SeaDAS is a scientific data visualisation, analysis, and processing software application for remote-sensing satellite data which derives from and is an extension of the BEAM Earth Observation Toolbox and Development Platform. SeaDAS includes support for the data products produced by NASA's Ocean Biology Processing Group (https://oceancolor.gsfc.nasa.gov). This support includes data processing capabilities to allow users to process data between the various levels (i.e. Level 0 through Level 3).
